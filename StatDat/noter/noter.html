1-stikprve T-test

Data:
Kontinuert responsvariabel [y] med ingen forklarende variabel \[y_1, ..., y_n\]

Statistisk model:
\(y_1, ..., y_n\) er uafhngige og alle normalfordelte med samme middelvrdi $_i = +e_i $ hvor \(e_i \sim N(0, \sigma^2)\) og samme spredning \(\sigma\).

De ukendte populationsparametre
Middelvrdi \(\mu\) og spredning \(\sigma\)

1-sidet Anova

Pointen med en varaiansanalyse er at vi kan pvise forskelle mellem grupper ved at samligne forskellige kilder til variation.

Data:
\(y_1, ..., y_n\) [Forklar var de reprsentere] fra k grupper med \(n_j\) observationer i gruppe j.

Statistisk model:
Vi antager at \(y_1, ..., y_n\) er uafhngige og normalfordelte \(\sim N(\mu_i, \sigma^2)\)

Dertil antager vi at middelvrdien \(\mu_i = \alpha_{g(i)} + e_i\) hvor \(\epsilon_i \sim N(0, \sigma^2)\) afhnger af gruppen. Med samme spredning \(\sigma\) i alle grupper.

De ukendte populationsparametre
Middelvrdien for alle grupper \(\alpha_g(1) ,..., \alpha_g(k)\) og den ukendte spredningen \(\sigma\)

Linr regression

Data:
Par \((x_1, y_1), ..., (x_n, y_n)\) bestende af kvantitative kontinuerte data, bde for vores responsvariabel [y] og den forklarende variabel [x].

Statistisk model:
\[ y_i = \mu_i + \epsilon \] hvor \(\epsilon_i \sim N(0, \sigma^2)\)

Vi antager \(y_1 ,..., y_n\) er uafhngihed, og at alle \(y_i\) er normaltfordelt med middelvrdi \(\mu_i = \alpha + \beta x_i\) (ret linje) med spredning \(\sigma\).

De ukendte populationsparametre
Skringen \(\alpha\), hldningen \(\beta\) og spredningen \(\sigma\) er alle ukendte parametre, som vi kan estimere: \(\hat{\alpha},\hat{\beta}, \hat{\sigma}\)

Data:
Par \((x_1, y_1), ..., (x_n, y_n)\) bestende af kvantitative kontinuerte data, bde for vores responsvariabel [y] og den forklarende variabel [x].

ANCOVA (Flere rette linjer)

Data De forklarende variabel bestr af en kontinuert variabel x [forklar X], og en kategorisk variabel grp [Forklar grp]. Respons variablen \(y_i\) er kontinuert

Statistisk model Additativ \[ y_i = \alpha_{\texttt{grp(i)}} + \beta \cdot x_i + \epsilon_i \]

hvor \(\epsilon_i \texttt{ iid. } N(0, \sigma^2)\)

Som det ses i formlen for middelvrdien angivet overfor, er effekten af x ikke afhngig af gruppen. Sledes for vi en model med flere parallele linjer med hldningskoefficent \(\beta\) og skring \(\alpha_{\texttt{grp(i)}}\)

Parameter Skringen per gruppe \(\alpha_{\texttt{grp(i)}}\), hldningen \(\beta\) og spredningen \(\sigma\) er alle ukende populationsparametre, som vi kan estimere.

Multiliner regression / Multipel liner regression

Modeller med \(d\geq2\) kvantitative forkl. variable. \[y_i = \alpha + \beta_1 x_{i1} + ... + \beta_d x_{id} + e_i\] Data
Tupler af data $(x_{11}, x_{11}, y_1), , (x_{n1}, x_{n1}, y_n) $ med forklarende variable [Beskriv xerne] og respons variabel [Beskriv y]

Statistisk model
Udgangspunkt i to forklarende variable: \[y_i = \alpha + \beta_1 x_{i1} + \beta_2 x_{i2} + e_i\] hvor \(e_i \sim N(0, \sigma^2)\)

De ukendte populationsparametre
De partielle hldninger \(\beta_1\), \(\beta_2\), skring med y aksen \(\alpha\) og spredningen \(\sigma\) er populationsparametre som vi kan estimere: $, , , $

Multikollinearitet Multikollinearitet opstr hvis \(x_1\) og \(x_2\) er afhngie, sledes at det bliver svrt at adskille effekten af dem.

Vi skal altid vre opmrksom p kollinearitet, og tegn p dette kunne vre: * Unaturlige estimater (Forkert fortegn) * Hverken \(\beta_1\) eller \(\beta_2\) er signifikante

Vi har den statistisk model \[y = \beta_0 + \beta x_{1} + \beta_2 x_2 + \epsilon \]

Men hvis der er liner afhngighed glder det at

\[ x_2 \approx a + b x_1 \]

Hvilket implicere \[ y \approx \beta_0 + \beta x_{1} + \beta_2 (a + b x_1) + \epsilon \] hvor \[ \epsilon \sim N(0, \sigma^2) \]

Binomial model

Data
Binr kategorisk responsvariabel

Statistisk model
\(Y \sim bin(n,p)\) med kendt n og ukendt p

Ukendte populationsparameter sandsynligheden p, spedningen og standard errror er ukendte parameter, som vi kan estimere \[ \hat{p} = \frac{\texttt{Antal succeser}}{\texttt{antal forsg}} = \frac{y}{n} \]

\[ sd(\hat{p}) =SE(\hat{p}) = sd(\frac{Y}{n}) = \sqrt{\frac{p(1-p)}{n}} \]

Beregning af sandsylighed

\[ P(Y=j) = \left( \begin{matrix} n \\j \end{matrix} \right) \cdot p^j \cdot (1-p)^{n-j} \] hvor \[ \left( \begin{matrix} n \\j \end{matrix} \right) = \frac{n!}{j!(n-j)!} \]

Eller med R:

dbinom(1, size=3, prob=0.6) ## P(Y=1)
## [1] 0.288
pbinom(2, size=3, prob=0.6) ## P(Y<=2)
## [1] 0.784
dbinom(0, size=3, prob=0.6) ## P(Y=0)
## [1] 0.064
1-dbinom(0, size=3, prob=0.6) ## P(Y>=1)
## [1] 0.936

Konfidensinterval

\[ \hat{p} \pm 1.96 \cdot SE(\hat{p}) = \hat{p} \pm 1.96 \cdot \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \]

Hypotese test

Vi tester alid hypotesen \(H_0 : p = p_0\)

\(Y \sim bin(n,p)\) og \(X \sim bin(m,q)\), som er uafhngige

Bestem t-teststrrelsen og p-vrdien for hypotesen i liner regression

I en liner regression med 87 datapunkter er hldningen blevet estimeret til 5.12 med en standard error p 3.20. Bestem t-teststrrelsen og p-vrdien for hypotesen om at den sande hldning er 0

beta = 5.12 #Hldning
SE = 3.2 # Standard error
n = 87 # Antal observationer
hypotese = 0 # hypotesen om at "den sande hldning" er 0


# Test af hypotesen H: beta=4
T = (beta - hypotese) / SE

# Beregning af p-vaerdi

Konklusion af p-vrdi givet f-fraktil og f-teststrrelse

Man vil sammenligne grupperne i en ensidet ANOVA med 100 observationer og fire grupper med et F-test. Man beregner F-teststrrelsen til 4.19, og 95%-fraktilen i F(3,96) er 2.70. Hvad kan vi sige om p-vrdien?

Da \[ \texttt{p-value} = P(F \geq F_{obs}) \]

har vi at siden F-teststrrelsen (4.19) er strre end fraktilsen (2.70) s er p-vrdien er lavere end 0.05 og vi kan afvise at grupperne har samme middelvrdi.

Konklusion af p-vrdi givet to f-fraktil og f-teststrrelse

Vi er i gang med at udfre et t-test og fr en observeret teststrrelse p T = 1.99. I den relevante t-fordeling er 97.5% fraktilen lig 2.14, mens 95% fraktilen er 1.76.

Da teststrrelse (1.99) ligger mellem 97.5% fraktilen (2.14) og 95% fraktilen er 1.76. (1.76) ligger p-vrdien mellem 0.05 og 0.10

  • Mindre end 97.5% fraktilen ==> Strre end 0.05
  • Strre end 95% fraktilen ==> Mindre end 0.10
