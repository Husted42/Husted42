# File manegement
Create a file
```
touch <filename>
```

Remove a file or dictonary
```
rm -f <filename>
rm -r <directoryname>
```

Move a file
```
mv <src> <dest>
```

File manager
 - Just open nautilus

# Package mangement
Install package
```
sudo dpkg -i <file.deb>
```

Solve missing dependencies
```
sudo apt-get install -f
```
